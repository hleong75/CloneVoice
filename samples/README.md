# Sample Files

Ce répertoire contient des exemples de fichiers pour utiliser CloneVoice.

## Fichiers

- `example.csv` : Exemple de fichier CSV pour la Version 1

## Comment utiliser

1. Placez vos fichiers audio (.wav, .mp3, etc.) dans ce répertoire
2. Les noms des fichiers doivent correspondre à la colonne `audio_id` du CSV
   - Exemple : `sample_001.wav` pour `audio_id = sample_001`

## Structure attendue

```
samples/
├── example.csv
├── sample_001.wav
├── sample_002.wav
└── sample_003.wav
```
